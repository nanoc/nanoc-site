As a helper for automatic menu generation won't be universally usable it is not planned to include one in nanoc's core. (See here)[[BR]]
But there are some mails on nanoc's mailing list regarding this topic. This wiki page was created to collect useful information about how to generate menus with nanoc's help.

The two links above show code to do this with old versions of nanoc. The code shown below is an adapted version of the code in the top link above.
See here for the post on the group from Denis about this.



<% home_page = @items.find { |i| i.reps[0].path == '/' } %>

    
    
  • <%= link_to(home_page[:title], home_page.reps[0]) %>
    • 
    <% home_page.children.each do |page| %>
        
  • <%= link_to(page[:title], page.reps[0]) %>
    • 
    <% end %>
