Breadcrumb Navigation


Breadcrumbs or breadcrumb trail is a navigation aid used in user interfaces. It gives users a way to keep track of their location within programs or documents. The term comes from the trail of breadcrumbs left by Hansel and Gretel in the popular fairytale. (Wikipedia)

Helps you to add such 'You are here: Home >> About >> Me' link lists.

This article describes usage for nanoc 3 which has a helper for breadcrumbs built-in. See [wiki:Helpers/BreadcrumbNavigationNanoc2] for usage with older versions.


Usage


nanoc 3.0 includes a breadcrumbs helper: it provides a #breadcrumbs_trail method that just returns the array of breadcrumbs (items) so you can display the breadcrumbs as you like.

Use the helper in your layouts/default.html for example:



...
<% unless @item.reps[0].path == "/" %>
<% breadcrumbs_trail.each do |b| %>
	<%= link_to(  b[:title] , b.reps[0].path ) if b %>
	<%= b != @item ? '»' : '' %>
	<%end%>

<% end %>
...




Hints


Don't forget to enable the helper in your helpers.rb. (Manual)


include Nanoc3::Helpers::Breadcrumbs
